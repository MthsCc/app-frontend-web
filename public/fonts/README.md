# Fontes Filson Pro

Para usar a fonte Filson Pro no projeto, você precisa adicionar os arquivos da fonte nesta pasta.

## Arquivos necessários:

- `FilsonPro-Regular.woff2`
- `FilsonPro-Regular.woff`
- `FilsonPro-Medium.woff2`
- `FilsonPro-Medium.woff`
- `FilsonPro-Bold.woff2`
- `FilsonPro-Bold.woff`

## Onde obter:

A Filson Pro é uma fonte comercial disponível em:
- MyFonts: https://www.myfonts.com/products/filson-pro-complete-family-package-440146
- YouWorkForThem: https://www.youworkforthem.com/font/T5233/filson-pro

## Nota:

Se você não tiver os arquivos da fonte, o navegador usará as fontes de fallback (Segoe UI, etc.) até que os arquivos sejam adicionados.
